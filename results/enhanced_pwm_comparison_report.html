Quality Assessment Guidelines

Information Content Standards for CTCF PWMs:

  • Excellent: >15 bits total, >0.05 bits average per position, 2-5 conserved positions
  • Good: 10-15 bits total, 0.03-0.05 bits average per position, 1-2 conserved positions
  • Fair: 5-10 bits total, 0.02-0.03 bits average per position, 0-1 conserved positions
  • Poor: <5 bits total, <0.02 bits average per position, 0 conserved positions

Statistical Significance Interpretation:

  • p < 0.001: Highly significant improvement over null model
  • p < 0.01: Significant improvement over null model
  • p < 0.05: Marginally significant improvement over null model
  • p ≥ 0.05: No significant improvement over null model

Effect Size Interpretation (Cohen's d):

  • |d| ≥ 1.2: Very large effect
  • 0.8 ≤ |d| < 1.2: Large effect
  • 0.5 ≤ |d| < 0.8: Medium effect
  • 0.2 ≤ |d| < 0.5: Small effect
  • |d| < 0.2: Negligible effect

Methodology

PWM Quality Metrics:

  • Total Information Content: Sum of information content across all positions
  • Average Information Content: Mean information content per position
  • Conserved Positions: Number of positions with >1 bit information content

Null Models:

  • Random: Random sequences matching overall base composition
  • Shuffled: Original sequences with shuffled base order
  • Position Shuffled: Sequences with position-wise shuffled bases

Statistical Testing:

  • P-values: Empirical p-values based on null distribution comparisons
  • Effect sizes: Cohen's d calculated as (observed - null_mean) / null_sd
  • Significance threshold: p < 0.05
