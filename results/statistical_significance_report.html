Methodology

Null Models

  • Random: Random sequences with matched overall base composition
  • Shuffled: Sequences with shuffled order, preserving individual composition
  • Position Shuffled: Position-wise shuffled sequences

Statistical Tests

  • p-value: Empirical p-value based on null distribution
  • Effect Size: Cohen's d = (observed - null_mean) / null_sd
  • Significance threshold: p < 0.05

Effect Size Interpretation

  • Negligible: |d| < 0.2
  • Small: 0.2 ≤ |d| < 0.5
  • Medium: 0.5 ≤ |d| < 0.8
  • Large: 0.8 ≤ |d| < 1.2
  • Very Large: |d| ≥ 1.2
